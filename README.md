# Sentiment Analysis using NLP

This project implements a sentiment analysis model using classical NLP and machine learning techniques.

## Tech Stack
- Python
- Scikit-learn
- NLTK
- TF-IDF
- Logistic Regression

## Dataset
TSV-based sentiment dataset with labeled text samples.

## How to Run
```bash
pip install -r requirements.txt
python src/train.py
python src/predict.py
